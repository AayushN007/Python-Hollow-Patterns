# Python Hollow Patterns

## 📘 Overview

This repository contains a collection of **hollow star and number patterns** implemented in Python using nested `for` loops and conditional statements. The programs demonstrate how to create hollow geometric shapes and boundary-based patterns by controlling the placement of characters and numbers.

These pattern programs are commonly practiced in Python laboratories, coding assignments, and technical interviews to improve logical thinking and nested loop concepts.

---

## 📂 Patterns Included

### 🔹 1. Hollow Square with Cross and Plus

Prints a hollow square containing:

* Outer border
* Main diagonal
* Secondary diagonal
* Horizontal center line
* Vertical center line

---

### 🔹 2. Hollow Right-Angled Star Triangle

Prints a hollow right-angled triangle using the `*` symbol, displaying only the borders.

---

### 🔹 3. Hollow Number Triangle (Increasing)

Prints a hollow triangle using increasing numbers along the boundaries.

---

### 🔹 4. Hollow Number Triangle (Decreasing)

Prints a hollow triangle using decreasing numbers from the largest value.

---

### 🔹 5. Number Boundary Pattern

Prints a square-like number pattern where numbers appear only along specific boundaries and diagonals.
